#MIKROTIK

Network Address Translater (NAT) berfungsi untuk membungkus IP private dengan IP publik
=> Kita mengakses facebook dengan ip private dari hp, maka log yang tercatat adalah ip publik dari gateway (UM)
.
Srcnat berfungsi unuk mengirim paket dari router ke internet (sudut pandang mikrotik)
dstnat berfungsi untuk menerima paket dari internet ke router mikrotik (sudut pandang mikrotik)
.
Dalam pemilihan frequensi pada mikrotik pilihlah urutan ke 1,6,11 karena tidak saling tindih khususnya dipermukiman yang padat
.
Perbedaan WPA-PSK, WPA2-PSK, WEP Proteksi
=> WPA-PSK berfungsi untuk mengamankan jaringan secara langsung
=> WPA2-PSK teknologi yang lebih secure dari WPA-PSK, memiliki password dengan panjang karakter minimal 8 character serta mengamankan jaringan secara langsung
WEP Proteksi berfungsi untuk mengamankan jaringan wireless dari balik layar, wifi terlihat open namun memerlukan authentication dengan server. Kelebihan dari system ini server dapat membuat password lebih dari satu tergantung kebutuhan user.
.
Interface yang berdiri tegak dalam pemilihan interface router artinya memiliki koneksi atau terhubung
Interface yang miring dalam pemilihan interface router artinya tidak memiliki koneksi atau tidak terhubung
.
Mode Wireless pada mikrotik :
=> Mode AP Bridge, berfungsi untuk menyebarkan jaringan dan client yang terhubung dengan jaringan tersebut dinamakan station
=> Mode station, berfungsi untuk menangkap sinyal dari Access Point tertentu kemudian menyebarkannya kembali (penguat sinyal/repeter)
.
